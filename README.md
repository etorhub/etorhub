# Hey, I'm Etor

Senior frontend engineer in Barcelona. I build products end-to-end — from architecture and UI to tests and CI/CD — mostly in the TypeScript/React ecosystem.

Enthusiastic `if (true)`. Code is like art: you never finish it, you just stop working on it.

---

## What I do

I lean frontend but think full-stack: React, TypeScript, GraphQL, and Node.js on one side; Python when the problem calls for it on the other. I care about the whole delivery loop — unit and e2e tests, GitHub Actions, AWS deployments — not just the UI layer.

Currently at **Veriff** building identity verification products. Before that, I spent years on a B2B patent search platform at Ensemble IP — architecture, features, UI, and CI/CD from the ground up. Along the way I've shipped data-heavy dashboards, design-system-driven apps, and complex SPAs for companies like Netquest and Red Points.
